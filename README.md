# Bank Transaction Behavioral Clustering
Overview

This project analyzes bank transaction patterns using unsupervised learning to identify behavioral groups of customers based on transaction activity.

The clustering approach can help financial institutions understand spending behavior, risk patterns, and customer segmentation.


Key Features

> Data preprocessing and cleaning

> Exploratory Data Analysis

> Feature scaling

> Cluster visualization

> Cluster quality validation


Techniques Used

> Hierarchical clustering

> K-Means clustering

> Silhouette analysis


Models

> K-Means

> Hierarchical Clustering (Dendrogram analysis)


Libraries

> Python

> Pandas

> NumPy

> Scikit-learn

> Seaborn

> Matplotlib

> Fastcluster


Insights Generated

1. Identification of distinct transaction behavior groups

2. Visualization of customer clusters

3. Quantitative validation of cluster quality
